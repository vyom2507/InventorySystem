Inventory Management System (Object Oriented Application Development):

A robust JavaFX application for managing inventory products using a MySQL database. This project implements the Model-View-Controller (MVC) architecture for clean code organization.

Features
- User Authentication: Secure login system verifying credentials against a database.
- Role-Based Access: Admins have full CRUD capabilities; Users have read-only access.
- Database Integration: Real-time persistence using MySQL.
- Custom UI: Enhanced JavaFX interface with custom CSS styling.

Technologies
- Language: Java (JDK 21)
- GUI: JavaFX
- Database: MySQL
- Tooling: Eclipse IDE

How to Run
1. Clone this repository.
2. Import the project folder into Eclipse.
3. Ensure the MySQL database is running and DBConnect.java has valid credentials.
4. Run Main.java to launch the application.
